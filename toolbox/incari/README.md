# Incari

## Introduction

The **Incari** **Nodes** encompass several **Object** types that are the driving force of incorporating eye-catching and informative visuals within a *User Interface*. With options for **Animations**, **Media**, **Lists**, and more, there are many ways to pursue designs that fit the user's needs.
